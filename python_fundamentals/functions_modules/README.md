# Python - Functions & Modularity

## Description

This project introduces the concept of functions and modular programming in Python.

As programs grow, repeating the same logic becomes inefficient and difficult to maintain. Functions allow us to organize code into reusable blocks, while modules allow us to separate code into different files.

The main goal of this project is to understand how to define functions, return values, import code from other files, and control how Python files behave when executed directly or imported.

## Learning Objectives

By the end of this project, I should be able to explain:

- What a function is
- How to define a function in Python
- How to use parameters and return values
- The difference between `print` and `return`
- How execution flows inside and outside functions
- How Python executes a file
- How to import functions from another file
- How to import variables from another file
- What `__name__ == "__main__"` means
- Why `if __name__ == "__main__":` is useful
- How to organize code across multiple files
