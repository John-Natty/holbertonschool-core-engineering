# Python - Environment & First Programs

## Description

This project introduces the basics of running Python programs and understanding the Python development environment.

The main goal is to understand how Python code is executed in two different ways:

- Interactively, using the Python interpreter, also called the REPL.
- By executing Python script files.

This project also introduces important development tools such as `pip`, `venv`, and `pycodestyle`.

## Learning Objectives

By the end of this project, I should be able to explain:

- How the Python interpreter evaluates expressions and statements.
- The difference between interactive execution and script execution.
- When Python displays output automatically in the REPL.
- The role of the `print()` function.
- How to create and execute a Python script.
- How to install Python tools using `pip`.
- Why global installations can cause conflicts.
- How virtual environments isolate project dependencies.
- How to check Python code style with `pycodestyle`.

## Requirements

- Operating system: Ubuntu 20.04 LTS
- Python version: Python 3.8.x
- Code style: `pycodestyle` 2.7.x
- All Python files must start with:

```python
#!/usr/bin/env python3
```

- All Python files must be executable.
- All files must end with a new line.
- No external libraries are allowed unless explicitly requested.
- Output must match the expected format exactly.
